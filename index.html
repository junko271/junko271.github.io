Overview

This project, “Metagram”, is a social media mock application that allows the user to simulate their regular social media behavior; the goal of this mock app is to give a report on how the user uses social media. It is well known that too much investment in social media can make the user unhappy, especially if the user has a large digital presence; users with a large presence tend to have larger insecurities. Thus, this project, “Metagram”, is aimed to allow social media users to reflect on their digital behavior once again.“Metagram” uses the prefix “meta” because the prefix means "giving explicit awareness to itself", or "referring to oneself". Since the goal of this project is to give the user a better understanding of themselves by giving insight on how they might be using social media, the prefix is included in the project title.

To the right is an example run of the project.
“Metagram” has options to input date of birth, location, pictures, school/work, and email (when creating an account). All of this information is private information that users should be careful of putting on the Internet. Another feature is that users can choose the duration between each post, representing their preferred digital presence, and also choose posts and captions based on comments they get. Metagram purposefully gives more than one negative feedback on each post to see if the user is sensitive to how they are perceived on the digital platform. After 3 posts, the user can view their report on their own digital behavior and tendencies based on the choices they made.
